FurEver Home - Pet Adoption Portal

PROJECT OVERVIEW

The Pet Adoption Portal is a web-based application that allows users to browse and adopt pets. It provides basic functionality for user registration, login, and viewing available pets.

OBJECTIVES

1. To develop a user-friendly pet adoption platform
2. To implement user registration and login functionality
3. To display pets dynamically from a database
4. To store user and adoption data

TECHNOLOGIES USED

1. Frontend: HTML, CSS
2. Backend: PHP
3. Database: PostgreSQL
4. Version Control: Git and GitHub
